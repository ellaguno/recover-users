Send personalized emails to inactive users with their previous posts and latest blog updates to re-engage them with your community.

== Description ==

User Reactivation Email is a plugin that helps keep your WordPress community active by sending personalized emails to users who haven't posted recently. Each email includes:

* A summary of the user's latest posts (up to 5)
* The blog's latest 5 posts
* A friendly message inviting them to participate again

Key features:

* Automatic weekly email sending
* Multi-language support
* Admin dashboard for configuration
* Email tracking
* Customizable email templates
* Responsive admin interface

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/user-reactivation` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the options in Settings > User Reactivation

== Frequently Asked Questions ==

= How often are emails sent? =

By default, emails are sent weekly. You can modify this frequency from the admin panel.

= Can I customize the email content? =

Yes, you can modify the email content from the admin panel.

= Does it support multiple languages? =

Yes, the plugin comes with English by default and includes Spanish translations. You can add more languages by creating additional .po files.
